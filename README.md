## パッケージ一覧

- comment_msgs : YouTubeコメントのメッセージ
- motion_socket : Webotsのモーションを解釈してcurlで送るノード
- youtube_comment : YouTubeコメントを取得するノード
- chatgpt_ros2 : ChatGPTのROS2パッケージ
